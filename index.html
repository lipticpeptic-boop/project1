Before → after: a quick refactor

The same button, cleaned up. Toggle to compare.

<!-- clickable div, inline styles, raw output -->
<div class="btn" onclick="submit()">
  <?php echo $name; ?>
</div>

<style>
  .btn{ color:#fff; background:#333 }
</style>
<!-- real button, escaped output, styles in CSS -->
<button class="btn" type="submit">
  <?= htmlspecialchars($name) ?>
</button>

/* stylesheet.css */
.btn:focus-visible{ outline:2px solid #F5B455 }
3 issues found
  • ✕A div can't be reached by keyboard or read as a button.
  • ✕Echoing $name raw invites XSS — it needs escaping.
  • ✕Inline styles & no focus state — hard to maintain, poor a11y.
