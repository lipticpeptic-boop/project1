# Персональное портфолио

Одностраничный сайт-портфолио, написанный с нуля на HTML, CSS и чистом JavaScript. Без фреймворков, без библиотек, без сборки — весь код лежит в одном самодостаточном файле `index.html`.

Центральный элемент — интерактивный резайзер вьюпорта: потяните за ручку (или используйте клавиатуру / пресеты устройств) и увидите, как встроенный макет перестраивается под мобильный, планшет и десктоп. Это живая демонстрация адаптивной вёрстки, а не просто слова о ней.

## Стек и технологии

- **HTML5** — семантичная, доступная разметка
- **CSS3** — CSS-переменные для темизации, Flexbox и CSS Grid для вёрстки, media-запросы и `clamp()` для «резиновой» типографики, градиенты, `backdrop-filter` и CSS-переходы
- **JavaScript (vanilla)** — без зависимостей; построен на браузерных API IntersectionObserver, Pointer Events и `matchMedia`
- **Google Fonts** — Space Grotesk (заголовки), Manrope (основной текст), JetBrains Mono (код и подписи)

## Возможности

- **Интерактивный резайзер вьюпорта** — тяните за ручку, используйте стрелки на клавиатуре или выбирайте пресет (Phone / Tablet / Desktop); мини-макет внутри перестраивается в реальном времени, а подпись брейкпоинта обновляется вместе с ним
- **Живой индикатор брейкпоинта** — показывает реальную ширину страницы и текущий брейкпоинт (XS–XL) и обновляется при изменении размера окна
- **Панель «до / после»** — переключатель с небольшим примером рефакторинга: семантичный HTML, экранированный вывод PHP и корректное состояние фокуса
- **Адаптивность и кросс-браузерность** — «резиновые» макеты, которые держат форму от телефона до десктопа в Chrome, Firefox, Safari и Edge
- **Доступность** — ARIA-слайдер с полным управлением с клавиатуры, видимые состояния `:focus-visible`, семантичная структура и поддержка `prefers-reduced-motion`
- **Прогрессивное улучшение** — анимации появления при скролле, мобильное меню и подсветка активного пункта навигации — всё с корректными запасными вариантами, если API недоступны

## Запуск локально

Никаких сборщиков не нужно. Склонируйте репозиторий и откройте файл:

```bash
git clone https://github.com/lipticpeptic-boop/project1.git
cd project1
```

Затем откройте `index.html` прямо в браузере или запустите локальный сервер:

```bash
# Python 3
python -m http.server 8000
# затем откройте http://localhost:8000
```

При первом открытии нужен интернет, чтобы подгрузились шрифты Google Fonts.
